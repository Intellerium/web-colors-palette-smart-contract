# Web Colors Palette

_MIT_ _License_

Unique colored pixels NFT collection of representing the Web 1.0 Colors Palette.

Developed on solidity for Ethereum Virtual Machine.

See more about [Web Colors](https://en.wikipedia.org/wiki/Web_colors)

## How to

### Prepare project

`$> npm install`

### Build contract

`$> npm hardhat compile`

### Test contract

`$> npm hardhat test`
